    Chloride

    Command-line tool to convert js files to Clojure/Chlorine ones.

    Get Chloride

    You can build chloride with:

    lein uberjar

    or download the jar file.

    Usage

    Start esparse-server, then:

    java -jar chloride-0.1.0-standalone.jar -r 2000 some-dirs-or-files
